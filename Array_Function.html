Object based Array Functions The array object lets you store multiple value in a single variable. It stores a fixed-size sequential collection of elements of the same type. An array is used to store a collection of data, but it is often more useful to think of an array as a collection of variables of the same type Syntax to create an array object: var Array_Name = new Array("Item1", "Item2", , ,) Example: var fruits = new array("Apple", "Banana", "Mango") THe array parameter(items) is a list of strings or integers. When you specify a single numeric parameter with the array constructor. The maximum length allowed for an array is 4,294, 965,295 Javascript supports the following object based array functions 1. Contact(): It returns new array comprised of this array joined with other arrays and/or value Syntax: array1.contact(array2); Example: Array contact method Output: alphanumeric: a, b, c, 1, 2, 3 2. Join(): Javascript array Join() method joins all the elements of an array into a string Syntax: array.join(separator); Example: Array Join method Output: str: First, Second, Third str: First, Second, Third str: First+Second+Third 3. POP(): Javascript array pop() method removes the last element from an array and returns that element Syntax: array.pop(); Example: Array pop method Output: element is:9 element is:4 4. The first element becomes the last and the last becomes the first Syntax: array.reverse(); Example: array reverse method Output: Reversed array is: 3 2 1 0 6. Array sort() method: Javascript array sort() method sorts the elements of an array Syntax: array2 = array.sort(array 1); Example: Array sort method Output: sorted array: banana, mango, orange, sugar 7. array slice() method: Javascript array slice() method extracts a section of an array and returned a new array Syntax: array.slice(begin index number, end); Example: array slice method Output: arr.slice(1,2): Mango arr.slice(1,2): mango, banana
