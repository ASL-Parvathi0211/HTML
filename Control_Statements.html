Control Statements Control statements are used to execute the statements based on the conditions is satisfied. Javascript supports the following control statements i. Conditional Control Statements These statements are used to execute based on the condition is satisfied. Javascript supports the following conditional control statements a. if statement In the if statement, if the condition is true it will execute immediate statement and condition with next statements Syntax: if (condition) statement; next statements; Example: if statement b. if_else statement In the if else statement, if the condition is true it will execute true block of stattements otherwise false block of statements and continue with next statements Syntax: if (condition) { true block of statements; } else { false block of statements; } next statements; Example: if else statement c. else_if statement: A if statement containes number of alternative if statement is called else_if lader statement Syntax: if (condition 1) statement 1; else if (condition 2) statements 2; ; ; ; else statement n; next statements; Example: else_if statement d. Switch Statements Switch statement executes directly particular case of the switch statement otherwise it will execute default statement Syntax: Switch (variable) { case 1: statement 1; break; case 2: statement 2; break; case 3: statement 3; break; ; ; default: default_statement; } Example: Switch statement ii. Looping Statement Javascript supports the following loop statements a. while loop It is a entry-controlled looping statement It is also called Pre-Test condition It will execute until the condition is false Synatax: while (condition) { looping statements; Increment/Decrement Variable; } Next statements; Example: while loop statement Example 2: Factorial value b. do_while loop It is exist controlled looping statement It is also called post test condition It will execute untill the condition is false Even the first time condition false, looping statements are executed one time Example: do_while loop c. for loop statement For loop statement consists of three expressions Initialization, Condition, Increment/Decrement variable Synatax: for (Initialization; condition; Increment/Decrement variable) { looping statement } next statemetns Example: For loop statement
