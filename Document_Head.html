# Explain in detail about HTML Document Head The head of the document contains information to be used by servers and browsers. It contains the title of the document which will be displayed at the top of the browser and optionally a list of keywords, ad description of the document, any files to be linked into the document and information about how the document was prepared HTML head Elements: | Tag | Description | |--------------------|------------------------------------------------------------------------| | | Defines information about the document | | | Defines the title of a document | | | Defines a default address or a default target for all links on a page | | | Defines the relationship between a document and an external resource | | | Defines metadata about an HTML document | | | Defines a client-side script | | | Defines styles information for a document | ===================================================================================================================================================================================================================================================================================================================================================== Example Program Online Shopping website
