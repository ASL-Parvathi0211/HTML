# escape Sequence Characters HTML entities are used to display reserved or special characters that would otherwise be interpreted as part of HTML syntax. They all start with "&" and end with ";". &, <, >, ",  , © therse are character escape sequences which are required if we want to display characters as HTML uses as control sequences.
