Exception Handling In Javascript a warning or error called execution which arises during the program execution. There are three types of errors in programming 1. Syntax errors: According to computer science a syntax error in the syntax of a sequence of characters or tokens that is included to be written in a particular programming language or it is also the compile time error. Tf the syntax is not correct then it will give an error message 2. Logical errors: It is the most difficult errors to be tracked to be traced as it is the error on the logic part of the coding or logical error and terminate abnormally 3. Runtime errors: A runtime error is an error that occurs during the running of the program also known as exception Javascript can implements exception handling using try, catch, finally and throws key words The try statements lets you test a block of code for errors. The catch statement lets you handle the errors. The throw statement lets you create custom errors. The finally statement lets you execute code after try and catch of the result Syntax: Example: Exception Handling Using of try-catch blocks Example 2: Using try-catch blocks
