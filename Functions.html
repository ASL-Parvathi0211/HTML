Functions A javascript function is a block of code to perform a particular task. A javascript function is executed when calls it A javascript name can contain letters (alphabets), digits, unferscore and dollar signs. In the parenthesis may contain parameter names separated by commas (parameter1, parameter2, -----) The code to be executed by the function is placed inside curly brackets: {} Syntax: function function_name(parameter1, parameter2) { code to be executed - - - - - - - - - - } Example: Javascript functions
