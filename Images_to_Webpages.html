# How can you set the images to your web pages Images play an important role in web design. They make pages visually appealing and help convey information more effectively. In HTML, images can be added inside the page content, or used as backgrounds through CSS. ============================================================================================================================================================================================================================ Image formats: There are several types of image format. They are Different image formats are supported in HTML. Each serves a different purpose: | Format | Full Form | Description | |---------------|----------------------------------|---------------------------------------------------------------------------------------| | JPEG (or JPG) | Joint Photographic Experts Group | Common for photos; supports millions of colors but no transparency. | | PNG | Portable Network Graphics | Supports transparency; ideal for logos, icons, and sharp-edged graphics. | | GIF | Graphics Interchange Format | Supports animation and limited colors; used for simple animations. | | BMP | Bitmap | Large file size; rarely used on modern web pages. | | SVG | Scalable Vector Graphics | XML-based format; best for logos and illustrations; scales without quality loss. | | WebP / AVIF | — | Modern formats that give better quality with smaller size; preferred for performance. | --> To set the image on the webpage we use tag Syntax: ============================================================================================================================================================================================================================ IMG Tag Attributes The tag is used to embed an image into an HTML document.
