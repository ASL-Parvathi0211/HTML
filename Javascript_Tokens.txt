Javascript Tokens
    Smallest individual words or symbols are called tokens. These tokens Javascript can understand when javascript is interpreted 

1. Identifiers
    identifiers are nothing but variables, methods or object in Javascript. They consist combination of characters and digits. You must begin all identifiers with either a letter or underscore 

    Rules for declaring Identifier:
    --> The first character in an identifier name must be an alphabet or underscore
    --> The identifier name should not be any key-word of the Javascript
    --> No commas or blanks are allowed within an identifier name
    --> No special symbol other than underscore(-) allowed 

2. Keywords
    Keywords are the words whose meaning already pre-defined by programming language. In Javascript they perform unique functions such as declaring new variables and functions 
    Keywords, which are built into Javascript are always available for use by the Programmers
    break, if, this, function, continue, in, true, return, else, int, var, false, new, while, for, null, with 

3. Literals
    Literals are nothing but a constnt values. These values do not change during the execution of your scripts.
    a. Integer Literals 
       Var max_marks = 100;
       var max_marks = 0100 (octal bases)
       Var max_marks = 0x100 (Hexa Decimal_base 16) 

    b. Floating Point Literals 
       Var pi = 3.1432;
       Var rate_of_interest = 8.50;

    c. Boolean Literals 
       Javascript implements Boolean data type and therefore it support true or false. Boolean value 1 for true and 0 for false 

    d. String Literals 
       A string literals is one or more characters enclosed in double (" ") or single(' ') quotes
       Example: Var name = "ASLParvathi"
                       or 
                Var name = "ASLParvathi"
                var college name = "Texas Tech University"

4. Operators
    An operator is a symbol that perform a specific operation. The data elements upon which an operator will perform a particular operation are called "operands" 
    a. assignment Operators: =, +=, -=, *=, /=, %= 
        Incremental and decremental operator: ++, --
        Conditional operator: variable = Expression1?Expression2:Expression3
        Logical Operators: && (AND), ;;(OR), !=(Not Equal to)
        Bitwise operators: &(Bitwise_AND), ;(Bitwise_OR), ^(Bitwise_XOR)
        Shift Operators: <<(Left Shift), >>(Right Shift)
