Rollover Buttons A rollover button is a dynamic web button that changes appearance depending on the location of the user's mouse pointer. It contains three states normal, over and down. The normal state appears when your mouse is off button, the over state applies when your mouse rolls over the button and the down state applies to when you click on the button. Javascript is commonly used to create these rollover effects, using the "on mouse over" event handler to trigger the rollover graphic and "on mouse out" event handler to return the graphic to its original state.
